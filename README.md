# LinuxCommandPractice

### Viewing hidden files and directories
- `ls -a`
- `-a` flag shows all files

### Wildcards
- `?` represents any one character
- `*` represents any amouny of any characters
- `[]` specifies a range
- `{!}` 
- `[!]` is the same as `[]` but acts as a NOT operator
- `\` is the escape character
